視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種讓網站背景比前景移動得更慢的網頁設計技巧。當訪客往下捲動畫面時，這會營造出 3D 視覺效果，讓畫面更具深度並 打造更沉浸的瀏覽體驗。視差建立在視覺錯覺上：人眼會覺得近處物體比遠處物體更大，因此遠處物體看起來移動得較慢。這種錯覺早已 被應用在各種媒材的視差效果中，創造逼真的臨場感。最早可以追溯到迪士尼《白雪公主》這類傳統動畫，以及像《超級瑪利歐》這類 電玩。隨著 CSS 與 HTML 的進步，視差效果逐漸演變成如今熟悉的網頁設計技術。

2007 年，網頁開發者 Glutnix 在部落格分享了最早的瀏覽器視差捲動實作之一，附上了支援 IE6 等瀏覽器的 JavaScript 與 CSS2 範例 與示範。然而直到 2011 年 HTML5 與 CSS3 普及後，這項技術才真正受到設計師青睞。擁護者認為這是擁抱網頁流動性的一種簡單方法； 設計師會利用視差背景來強化互動並改善整體體驗。不過，普渡大學在 2013 年的研究指出：「……雖然視差捲動增強了使用者體驗的某些 面向，但未必能全面提升整體使用者體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因如下：

  • 有助於說故事

    視差圖像非常適合用來說故事。優秀的設計就是有效溝通，當我們打造網站時，希望與訪客對話，講述產品、服務或品牌的迷人故 事。視差捲動可以讓訪客沉浸在獨特的線上體驗中，甚至能以單頁網站就完整呈現品牌或產品的故事，而不必跳轉多個頁面。

  • 提升使用者參與度

    視差圖像也能提升互動。降低網站跳出率是設計時的重要目標。跳出率指的是使用者只看一頁就按返回或關閉分頁的比例。高跳出率 會影響搜尋排名，因此任何能降低它的做法都值得嘗試。打造得宜的視差頁面能提供動態體驗，讓使用者願意停留更久。


想在網站加入視差捲動效果時，請留意以下建議：

  • 選擇性使用

    視差捲動用在特定元素時比整站全面使用更有力。可以只放在標題、頁首或首頁，重點是強化設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差捲動仰賴多種媒體與 CSS 形狀來營造深度。若圖片太大載入過慢，就會犧牲效能與體驗，因此一定要記得壓縮。

  • 善用色彩

    視差捲動不只關於前景與背景圖像，也包含色彩。背景可用較低飽和度的色調，前景則使用更鮮明的色彩，能營造更強烈的深度感。
